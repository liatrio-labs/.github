# Liatrio Labs

**Innovation Organization** | Open source projects from [Liatrio](https://www.liatrio.com/)

Liatrio Labs is Liatrio's open source innovation organization, focused on building tools and workflows that help engineering teams modernize their development practices and improve developer experience.

## What We Build

We create open source tools and workflows that support:

- **AI-Assisted Development**: Prompts, workflows, and tooling for AI-powered software development
- **Developer Experience**: Templates, automation, and best practices for modern software projects
- **DevOps Enablement**: Tools that help teams adopt modern DevOps practices and workflows

## Featured Projects

### [spec-driven-workflow](https://github.com/liatrio-labs/spec-driven-workflow)

Build predictable software with a repeatable AI-guided workflow. Prompt-driven approach to spec-driven development that keeps AI collaborators and human developers aligned.

### [slash-command-manager](https://github.com/liatrio-labs/slash-command-manager)

CLI tool and MCP server for generating and managing slash commands in AI coding assistants. Enables seamless integration of prompts as native commands.

### [ai-prompts](https://github.com/liatrio-labs/ai-prompts)

A curated collection of prompts for AI tools used in day-to-day software development activities, organized by use case.

### [open-source-project-template](https://github.com/liatrio-labs/open-source-project-template)

A battle-tested GitHub template repository with opinionated developer experience, quality gates, and CI/CD automation ready for customization.
